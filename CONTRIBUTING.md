# Contributing

Thanks for helping grow **awesome-midi-repos**. Contributions are welcome and kept intentionally simple.

***

## Quick steps
1. **Fork** this repository.  
2. **Edit** `README.md` in your fork.  
3. **Add** one line with the repository link in the exact format shown below.  
4. **Commit** your change.  
5. **Open a Pull Request** to `asigalov61/awesome-midi-repos` targeting the `main` branch.

## Exact entry format
Add exactly one line per repository using this minimal markdown format:

```markdown
- [repo_name by owner_name](https://github.com/owner/repo)
```
